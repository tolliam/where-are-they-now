# 📺 Where Are They Now?

An interactive map of UK infrastructure projects I worked on during two years at a major engineering consultancy (2016–2018) — and what became of them.

Some got built. Some got cancelled. Some are still stuck in planning purgatory.

## Projects

| Project | Location | Status |
|---|---|---|
| Cardiff Parkway | Cardiff, Wales | Pre-Construction |
| Railway Test Track Wales (GCRE) | Neath Port Talbot, Wales | Under Construction |
| A30 Chiverton to Carland Cross | Cornwall | Completed |
| A303 Stonehenge | Wiltshire | Planning Dispute |
| M4 Relief Road | Newport, Wales | Cancelled |
| Southsea Coastal Defence | Portsmouth | Under Construction |
| Manchester North West Quadrant (MNWQ) | Greater Manchester | Scaled Back |
| Temple Quarter Enterprise Zone | Bristol | Under Development |
| Gatwick Airport Station | West Sussex | Ongoing Improvements |
| Bristol Temple Meads Redevelopment | Bristol | Restoration Ongoing |
| Brabazon, Filton | Bristol | Under Construction |

## Running locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Built with

- [Streamlit](https://streamlit.io) — app framework
- [Folium](https://python-visualization.github.io/folium/) — interactive maps
- [streamlit-folium](https://folium.streamlit.app) — Folium integration for Streamlit
